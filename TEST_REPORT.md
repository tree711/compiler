### 测试2：嵌套 if-else 语句
**测试文件**: `test2_nested_if.c`
**测试内容**: 
```c
int a, b, c;
a = 10;
b = 20;

if(a > 5)
{
    if(b > 15)
    {
        c = 1;
    }
    else
    {
        c = 2;
    }
}
else
{
    c = 3;
}
```
**预期结果**: 
- 嵌套 if-else 正确处理
- 控制流正确：`a=10>5` 且 `b=20>15`，所以 `c=1`

**测试结果**: ✅ 通过
- 词法分析：识别 51 个 Token
- 语法分析：生成嵌套的 if-else AST
- 中间代码：生成 5 个基本块
- 代码生成：生成正确的条件跳转指令

### 测试3：嵌套 while 循环
**测试文件**: `test3_nested_while.c`
**测试内容**: 
```c
int i, j, sum;
i = 0;
sum = 0;

while(i < 5)
{
    j = 0;
    while(j < 3)
    {
        sum = sum + 1;
        j = j + 1;
    }
    i = i + 1;
}
```
**预期结果**: 
- 外层循环执行 5 次
- 内层循环执行 3 次
- 最终结果：`sum = 5 * 3 = 15`

**测试结果**: ✅ 通过
- 词法分析：识别 46 个 Token
- 语法分析：生成嵌套的 while AST
- 中间代码：生成 6 个基本块
- 代码生成：生成正确的循环跳转指令

### 测试5：比较运算符
**测试文件**: `test5_comparison.c`
**测试内容**: 
```c
int x, y, result;
x = 10;
y = 20;

if(x < y)
{
    result = 1;
}

if(x > y)
{
    result = 2;
}

if(x == y)
{
    result = 3;
}
```
**预期结果**: 
- 第一个条件成立：`10 < 20`，所以 `result = 1`
- 第二个条件不成立：`10 > 20` 为假
- 第三个条件不成立：`10 == 20` 为假

**测试结果**: ✅ 通过
- 词法分析：识别 42 个 Token

- 中间代码：生成 9 个基本块
